# uC/TELNETs

µC/TELNETs is Micrium's TELNET (Terminal Network) server. It features:

* Delivered with complete 100% ANSI C source code and in-depth documentation.
* Easy configuration
* Simple API

µC/TELNETs  is a component of µC/TCP-IP.
